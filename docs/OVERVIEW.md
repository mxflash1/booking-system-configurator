# 🚀 Booking Website Business - Quick Overview

## What This Is

A **Smart Template System** for building custom booking websites for barbershops and salons. You manually build each site using Cursor AI assistance.

## The System

1. **Client Landing Page** → Clients order websites
2. **Admin Dashboard** → You view orders
3. **Smart Workflow** → You build with Cursor AI help
4. **Deploy** → Sites go live

## Quick Commands

```bash
npm run help                    # Show all commands
npm run new-client <name>       # Create new project
npm run deploy <name>           # Deploy project
```

## Key Files

- `client-files/index.html` - Where clients order (SHARE THIS)
- `admin-files/admin-dashboard.html` - Your order dashboard
- `docs/SMART-TEMPLATE-WORKFLOW.md` - Complete workflow guide
- `docs/FIREBASE-SETUP-GUIDE.md` - Firebase setup (one-time)
- `.cursorrules` - Teaches Cursor your system

## Workflow (50 min per client)

1. Client submits order → You see in dashboard
2. Click "Start Building" → Get instructions
3. Run: `npm run new-client <name>`
4. Open in Cursor, update client-config.json
5. Tell Cursor: `@workspace Apply client-config.json`
6. Test locally, deploy to Firebase
7. Deliver to client!
